# 🌿 EcoWaste - Waste Management System (PHP Version)

A full-stack **waste collection and management web application** built using **PHP, MySQL, HTML, CSS, and JavaScript**.  
The system allows **users to schedule waste pickups, drivers to manage collections, and admins to monitor and manage the system**.

🌐 **Live Demo**  
http://ecowaste.infinityfreeapp.com

---

# 🎯 Overview

EcoWaste is designed to **simplify waste collection management** by connecting **citizens, waste collection drivers, and administrators** through a centralized system.

The platform enables:
- Easy **waste pickup scheduling**
- **Driver assignment and tracking**
- **Administrative monitoring and management**
- **Notification system for updates**

# 🔐 Authentication

The system supports **three types of login portals**:

- User login
- Driver login
- Admin login

Security features include:
- Password hashing
- Session-based authentication
- Input validation
- Basic SQL injection protection

---

# 📊 Database Structure

The system contains **6 main tables**:

| Table | Description |
|-----|-----|
| users | Stores registered users |
| drivers | Stores driver accounts |
| pickups | Waste pickup requests |
| notifications | System notifications |
| contact_messages | Contact form messages |
| sessions | User session data |

Features:
- Relational database structure  
- Indexed queries  
- Soft delete implementation
